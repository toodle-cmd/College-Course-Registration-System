## Features

- Student registration form
- Date and time display
- Name and personal details collection
- DOB validation
- Email validation
- Gender selection
- Phone number validation (10 digits)
- Course selection
- Branch/Specialization selection
- Registration summary
- Submit confirmation

  ## Available Courses

- B.Tech
- BCA
- B.Com
- BBA
- Law

  ## How to Use

1. Start a new registration.
2. Enter student details.
3. Enter a valid DOB.
4. Enter a valid email address.
5. Select gender.
6. Enter a valid 10-digit phone number.
7. Choose a course.
8. Select a branch/specialization.
9. Review the registration summary.
10. Type **yes** to submit the form.

- DOB format validation
